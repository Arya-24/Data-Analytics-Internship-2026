# Sales Forecasting using Time Series Analysis

## Project Overview
This project focuses on forecasting future retail sales using historical transaction data from the Superstore dataset. 

Time-series forecasting enables businesses to:
- Improve inventory planning
- Optimize marketing strategies
- Forecast revenue
- Prepare for seasonal demand fluctuations

## Dataset
- Superstore retail dataset (2019–2022)
- 9,994 transaction records
- Aggregated into 48 monthly observations

## Techniques Used
- Data Cleaning and Preprocessing
- Monthly Sales Aggregation
- Time-Series Visualization
- Prophet Forecasting Model
- Train-Test Split (36 months train, 12 months test)
- Model Evaluation (MAE, RMSE)

## Model Performance
- MAE: ~10,424
- RMSE: ~13,925

The model effectively captures:
- Long-term upward trend
- Strong yearly seasonality
- Recurring high-demand periods

## Key Insights
- Sales demonstrate steady growth from 2019 to 2022.
- Clear seasonal spikes occur annually.
- Forecast predicts continued growth into 2023.
- Inventory and marketing planning should align with seasonal peaks.

## Business Applications
- Demand forecasting
- Revenue prediction
- Supply chain optimization
- Financial planning
